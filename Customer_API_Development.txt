
===========================
Customer API Development
===========================

=> We have to implement below functionalities in Customer API

	1) Customer Registration with email sending

	2) Customer Login

	3) Reset Pwd

	4) Retrieve Customer Using Email

	5) Update Customer Details

	6) Save Customer Address

	7) Retrieve Address Using Address ID

	8) Update Address

	9) Retrieve All addresses of customer using customer id

	10) Delete Address using address id

Note: We need to implement Spring Security with JWT here.


====================
Development Process
====================

Step-1 : Create Project with required dependencies

			a) web-starter
			b) data-jpa-starter
			c) mysql-driver
			d) lombok
			e) devtools
			f) model-mapper
			g) swagger
			h) mail-starter
			i) security-starter
			j) actuator-starter

Step-3 : Configure Data Source Properties in "application.yml" file

Step-4 : Create Entities & Repo interfaces

Step-5 : Create DTO classes to map request & response data

Step-6 : Create Mapper classes to map DTO data to Entity and vice versa

Step-7 : Create Service interface & impl classes

Step-8 : Create ApiResponse class to represent standard response structure

Step-9 : Create Rest Controller to handle request and response

Step-10 : Run the API and test it using Postman or swagger
